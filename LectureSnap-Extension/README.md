# LectureSnap Bridge Extension

This extension enables the connection between YouTube and the LectureSnap app.

## Installation

1. Open Chrome/Brave and go to `chrome://extensions`.
2. Enable **Developer Mode** (top right).
3. Click **Load Unpacked**.
4. Select this `extension` folder.

## Usage

1. Open a YouTube video.
2. Open your LectureSnap app (e.g., `http://localhost:5173`).
3. On YouTube, press **Ctrl+Shift+S**.
4. The screenshot and timestamp will automatically appear in your LectureSnap app!

## Troubleshooting

- **No screenshot?** Ensure both the YouTube tab and LectureSnap tab are open.
- **Reloading:** If you change the code, click the "Refresh" icon on the extension card in `chrome://extensions` and reload both tabs.
